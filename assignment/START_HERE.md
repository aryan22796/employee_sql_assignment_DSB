## 🚀 How Students Use This

### First Time Through
1. Start with README.md (3 min)
2. Read PROBLEM_STATEMENT.md (15 min)
3. Study DATABASE_SCHEMA.md (25 min)
4. Follow LEARNING_GUIDE.md Step 1-4 (45 min)
5. Implement Steps 5-8 (60 min)
6. Check against SAMPLE_OUTPUT.txt (10 min)
7. Review with solution.sql (15 min)
8. Self-assess with RUBRIC.md (10 min)

### Debugging When Stuck
1. Check QUICK_REFERENCE.md section on errors
2. Use debugging queries in SAMPLE_OUTPUT.txt
3. Review relevant LEARNING_GUIDE.md step
4. Reference EXPLANATION.md for concepts
5. Compare to solution.sql

### Before Submission
1. Verify all 7 columns present
2. Check one row per employee
3. Validate calculations
4. Review code comments
5. Ensure explanation complete
6. Self-grade against RUBRIC.md
