## 🎓 Key Learning Concepts

This assignment teaches:

1. **Multiple JOINs** - Combining up to 5 tables
2. **Subqueries** - Advanced filtering with nested SELECT
3. **Date Functions** - TIMESTAMPDIFF, CURDATE
4. **String Functions** - CONCAT for readable output
5. **Aliases** - Clear table and column naming
6. **Historical Data** - Working with temporal databases
7. **SQL Performance** - Understanding query optimization
8. **Professional Practices** - Code comments, formatting
